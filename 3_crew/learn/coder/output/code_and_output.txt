```python
# Python code to calculate the first 10,000 terms of the series
# and multiply the total by 4.

# Initialize the total
total = 0

# Calculate the series
for n in range(10000):
    term = 1 / (2 * n + 1)  # Calculate the term (1/(2n + 1))
    if n % 2 == 0:  # Add for even index and subtract for odd index
        total += term
    else:
        total -= term

# Multiply the total by 4 as per the requirement
result = total * 4

# Print the result
print(result)
```
